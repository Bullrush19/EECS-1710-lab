Bennett Turnbull - Alphabet Exercise

required files:
	
- B_Turnbull_alphabet
- Circle
- 26 alphabet letters in datafolder

Instructions:
When ran, the program will generate an image of the letter A out of circles. to interact with the program you
first have to click into the window, then you can hit any letter button on the keyboard. The program should
then generate an image of the letter you just pressed out of circles.
